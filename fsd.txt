git clone *ссылка ssh* (берется с github - директория)
создаем рабочую область

git pull
переносим актуальную директорию с github

git push 
отправляем коммиты на github

git status
отображаются какие были сделаны изменения, закомичены или нет

git add *название файла или директории, которое есть в git status*
добавляем файл или директорию в индекс (то есть, предворительно делаем список
со всеми изменениями, которые закомитим и пошлем на гитхаб)

**выбрали допустим два файла, которые меняли и хотим это изменения записать
то есть, закомитить

git commit -m *название коммита, типо комментария что сделали*

git restore *название файла или директория*
отменяем действия с файлом или директорией, которые еще не были закомичены

git restore --staged *название файла или директории*
git log 
информация по всем коммитам, сортировка по новой дате сверху

git show *идентификатор коммита*
показывают всю информацию по коммиту

git diff
показывает изменения, которые еще не были добавлены в индекс

git blame *путь до файла* 
выводит файл и рядом с каждой строчкой показывает того, кто ее менял и в каком коммите

git grep *что ищем*
ищет совпадение с указанной строкой во всех файлах проекта
git grep -i *что-то ищем*
-i это без учета регистра
